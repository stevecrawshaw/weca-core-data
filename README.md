This is code to extract data from various web resources, including spatial data and environmental data, and load it into a DuckDB database. It is written in Python, it uses DuckDB, and it is intended to support a wide range of environmental assessment at a regional scale. 

The code has been written by a Python programmer who is at beginner or intermediate level. It requires comprehensive refactoring in order to follow best practises and make the process a truly reliable and operational product.
